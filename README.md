# Сказка 

Текстовая игра-викторина с элементами случайности, написанная на **Delphi**.

## О проекте

**"Сказка"** — это небольшая текстовая игра, в которой игроку задаются случайные вопросы. В зависимости от ответов программа составляет уникальный итоговый текст.

## Технологии

- **Среда разработки:** Delphi 
- **Платформа:** Windows

## Как играть

1. Запустите `Fairy_tale_Game.exe`
2. Отвечайте на появляющиеся вопросы (выбор вариантов или ввод текста)
3. После ответов программа сгенерирует уникальный текст-сказку


##  Запуск

### Готовый .exe
Просто дважды кликните `Fairy_tale_Game.exe` в Windows.
